About this project
Built to pass.
Built to share.
This is a suite of interactive study tools for the IIBA ECBA certification — built in the course of actually preparing for the exam. What started as a practice exam became a full study platform, developed collaboratively between a human and an AI in real time, on a road trip.
The Tools
📋
Mock Exam
50 questions across all six BABOK® v3 Knowledge Areas. Timed at 60 minutes with full answer review, explanations, and per-domain performance breakdown. Multiple exam versions available with no repeated questions.
📖
Study Guide
An interactive reference covering all six Knowledge Areas. Core tasks, key techniques, exam tips, and glossary terms per area. Tap any technique to reveal its definition inline.
🗂
Prep Hub
Quick-access dashboard with attempt history, exam target countdown, and one-tap navigation to all tools.
Co-developed by
This project was built collaboratively — the domain knowledge, direction, and feedback came from a human preparing for the ECBA exam; the code, content, and architecture came from Claude. Every feature was discussed, iterated, and refined in conversation: the question bank, the storage system, the study guide interaction model, the navigation, even this about page. It's a genuine example of what AI-assisted development looks like in practice — not a prompt, but a partnership.
